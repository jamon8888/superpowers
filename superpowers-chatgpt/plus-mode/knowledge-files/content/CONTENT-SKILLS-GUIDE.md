# Content Skills - Knowledge Files

This directory contains **4 comprehensive content skills** with full reference materials for ChatGPT Projects.

## How to Use

1. **Create a ChatGPT Project** for Superpowers
2. **Upload ALL files** from this `content/` directory to the Project knowledge base
3. Skills will automatically reference their support materials

## What's Included

### Skills (4 main files)
- `linkedin-thought-leadership.md` - LinkedIn content creation
- `ux-writing.md` - Interface copy and microcopy
- `ai-text-humanization.md` - English AI text humanization
- `humanisation-texte-ia-francais.md` - French AI text humanization

### Reference Materials (4 directories)

Each skill has supporting files in subdirectories:

**linkedin-thought-leadership/**
- `references/` - 3 framework guides (39KB)
- `assets/` - Personalization templates

**ux-writing/**
- `references/` - 3 guideline files (26KB)
- `templates/` - Error message template

**ai-text-humanization/**
- `master-prohibited-words.json` - **REQUIRED** (38KB, 1,643 lines)
- `docs/USAGE_GUIDE.md` - Complete usage guide

**humanisation-texte-ia-francais/**
- `mots-interdits-francais.json` - **REQUIS** (103 lignes)

## Required Files

**CRITICAL - Skills won't work without these:**
- ✅ `ai-text-humanization/master-prohibited-words.json`

**Highly Recommended:**

## ChatGPT Project Setup

**Recommended Project Structure:**

**Project Name:** "Superpowers Content Skills"

**Custom Instructions:**
```
I'm using Superpowers content skills for LinkedIn posts, UX writing, and AI text humanization. When I ask for help with content creation, check which skill applies and use it systematically. Reference the included materials as needed.
```

**Knowledge Files:** All files from `content/` directory

**Usage:**
- Ask: "Help me write a LinkedIn post about [topic]"
- ChatGPT will use linkedin-thought-leadership skill + references
- Ask: "Humanize this AI-generated text"
- ChatGPT will use ai-text-humanization skill + master-prohibited-words.json
